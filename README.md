# TimeBlocks

A modern, fast, and unified task management and planning app built with SolidJS.

## Features

- **Unified Dashboard**: View your Inbox, Kanban Board, and Calendar in one flexible workspace.
- **Inbox**: Quick capture for new tasks.
- **Kanban Board**: Drag and drop tasks between Todo, In Progress, and Done.
- **Calendar Planning**: Drag tasks to schedule them for the week.
- **Toggles**: Customize your view by toggling panels on/off.
- **Local First**: Data persists to LocalStorage.

## Usage

```bash
npm install
npm run dev
```

## Testing

```bash
npm test
```

## Deployment

Builds the app for production to the `dist` folder.

```bash
npm run build
```
